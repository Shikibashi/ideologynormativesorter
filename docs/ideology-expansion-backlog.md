# Ideology Expansion Backlog

## Methodology

This backlog follows the approved centroid-first curation plan. It is a review artifact, not an implementation patch: it does not define numeric centroids, label ids, question mappings, UI copy, validation tests, or scoring changes.

Source-list membership is evidence, not sufficient inclusion. A row is included when it appears useful for the sorter as either a distinct future result-label candidate or a grouped alias/subtype that clarifies an existing or candidate parent. Existing labels from `src/data/labels.ts` are used only as parent anchors and are not counted as net-new additions.

### Priority rule

- **P1**: Clear missing centroid. The candidate plausibly represents a distinct answer-pattern cluster across the existing axes.
- **P2**: Useful subtype or alias group. The candidate is better represented as a child, subtype, or synonym under an existing or candidate parent anchor.
- **P3**: Interesting but speculative. The candidate is recognizable or source-backed, but its distinct scoreable profile is weaker, narrower, or likely to require later debate.

Tie-break borderline cases toward P1 only when the candidate adds clear centroid distinctiveness; otherwise group as P2 or leave as P3.

## Exclusion and grouping notes

- Entries that map directly to current labels, such as Anarcho-Capitalism → Anarcho-Capitalist, Anarcho-Communism → Anarcho-Communist, Christian Democracy → Christian Democrat, Council Communism → Council Communist, Democratic Socialism → Democratic Socialist, Eco-Socialism → Ecosocialist, Ethnonationalism → Ethnonationalist, Geolibertarianism → Geolibertarian, Marxism-Leninism → Marxist-Leninist, Minarchism → Minarchist, Mutualism → Mutualist, Neoconservatism → Neoconservative, Neoreactionaryism → Neoreactionary, Social Democracy → Social Democrat, Syndicalism → Syndicalist, and Technocracy → Technocratic Centralist, should not be re-added as net-new rows.
- Narrow regime/personality labels such as Francoism, Pinochetism, Trumpism, Titoism, Hoxhaism, Kemalism, and Peronism are generally better treated as later subtype candidates unless a separate review finds durable centroid distinctiveness.
- Government-form abstractions such as Democracy, Oligarchy, Autocracy, Federalism, Presidentialism, Parliamentarianism, and Constitutionalism are usually institutional dimensions rather than result ideologies.
- Meme/meta or ultra-fictional entries such as Satirism, Soulism, Illuminatism, Death Worship, Puzzle Piece Politics, and Fully Automated Gay Space Communism should be excluded from the first pass unless the user later requests a novelty appendix.
- Single-issue or narrow cultural stances such as Climate Skepticism, Pink Capitalism, Men's Liberation, and Pacifist Feminism should generally be excluded or grouped under broader parents.
