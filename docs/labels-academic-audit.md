# Academic Audit of `src/data/labels.ts`

## Overall quality summary

The file is generally stronger than a simple left/right quiz: it distinguishes social democracy from democratic socialism, Marxism-Leninism from council communism, libertarian socialism from right-libertarianism, civic nationalism from ethnonationalism, and ecomodernism from ecosocialism and degrowth. The main weaknesses are precision and product presentation rather than wholesale conceptual failure: several labels are analytically defensible but too broad for a single centroid, a few names are nonstandard or sound polemical, and several niche/emerging labels need caution notes so users do not read them as canonical traditions. Axis placements appear broadly plausible at the family level, but labels marked “Needs narrowing” or “Use cautiously” should receive a centroid follow-up after naming/description changes because one centroid can currently flatten distinct variants.

## Prioritized fix list

### 1. Must fix before release
- Rename or narrow labels whose current wording can mislead users: “Fascist-Authoritarian” should not imply fascism is just generic authoritarianism; “Theocrat” should be framed as theocracy/theocratic politics; “Revolutionary Collectivist” should specify whether it means revolutionary state socialism.
- Add caution/scope notes for Zionism, Hindutva, Juche, Islamic Democracy, Eco-Fascism, Neoliberalism, and Accelerationism because academic usage is contested, internally diverse, or often polemically abused.
- Do a centroid follow-up for any label split or rename, especially Zionism, Islamic Democracy, Accelerationism, Neoliberalism, Eco-Authoritarianism, Cyberocracy, and Techno-Anarchism.

### 2. Should fix for academic precision
- Keep social democracy, democratic socialism, state socialism/Marxism-Leninism, council communism, syndicalism, market socialism, and libertarian socialism separate in names, help text, and centroid placement.
- Keep anarcho-capitalism, left-wing market anarchism, mutualism, individualist anarchism, and socialist anarchism separate; add a note that anarcho-capitalism is contested within anarchist studies.
- Clarify conservative liberalism vs liberal conservatism if both are displayed, because they are overlapping but not identical emphases.
- Mark cyberocracy, neoreaction, panarchism, techno-anarchism, and participism as niche/emerging rather than canonical traditions.

### 3. Useful UI/help-text improvements
- Add `academicUsageNote`, `commonUsageNote`, and `cautionNote` fields for labels with contested usage.
- Show “thin ideology/style” language for populism so users do not mistake populism for a complete doctrine.
- Add plain-language tooltips distinguishing classical liberalism, social liberalism, neoliberalism, libertarianism, and conservatism.
- Add help text for “state capitalism” and “corporatism” if those terms are introduced later; they are not current top-level labels in `labels.ts`.

### 4. Labels that are accurate and should be left alone
- Strong “keep” candidates: Democratic Socialist, Market Socialist, Social Democrat, Mutualist, Marxist-Leninist, Council Communist, Syndicalist, Anarcho-Communist, Minarchist, Agorist, Civic Nationalist, Ethnonationalist, Classical Liberalism, Ordoliberalism, Social Liberalism, Communitarianism, Republicanism, Distributism, Libertarian Socialism, Deep Ecology, Radical Democracy, Maoism, Trotskyism, World Federalism, Guild Socialism, Multiculturalism, Bioregionalism, Democratic Confederalism, Paleoconservatism, One-Nation Conservatism, and Liquid Democracy.
